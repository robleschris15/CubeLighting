--------- Specular Lighting : Chris Robles, Erick Lagunas, Jake Whitebread : CST-310 ----------

Contents:
- Docx file with documentation of the project
- Folder with main code files
- Executable file (cubes)

Code Execution - cpp file: 
- Go to the SourceCode folder with the commandline 
- Compile and run the desiered code with the following command 
     = g++ (name of file to be compiled) -o (output file name) glad.o -lglfw -ldl
     = ./(output file name)

Code Execution - Executable file: 
- Go to the SourceCode folder with the commandline 
- Execute with the following
     = ./cubes

Operation:
- After execution, the window produced can be closed with the Esc key
